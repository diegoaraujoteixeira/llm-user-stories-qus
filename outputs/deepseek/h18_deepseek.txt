As a developer, I want to use "Go To Declaration" (F3 or Ctrl+Click) on a symbol (e.g., variable, function, class, or method) to instantly navigate to its definition location so that I can quickly understand code flow without manually searching through files.

As a developer, I want to use "Find References" to see all occurrences of a selected element across the current file and the entire project, with results grouped by file and line number, so that I can assess the impact of changes and safely refactor my code.

As a developer, I want to use "Open Type" (Ctrl+Shift+T) to quickly find and navigate to a specific class, module, or interface across all project dependencies by typing its name with partial matching and camel-case suggestions so that I can jump to any type without manually browsing the project tree.

As a developer, I want the editor to support hyperlink navigation on imported symbols, require/import statements, and file paths (e.g., require('./helper'), import {x} from './lib') so that I can Ctrl+Click to open referenced files directly from the editor.

As a developer, I want navigation history (back/forward navigation) with keyboard shortcuts so that I can easily return to my previous editing location after jumping to declarations or references, maintaining my workflow context.
