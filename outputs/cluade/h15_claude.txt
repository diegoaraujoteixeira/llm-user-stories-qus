As a developer, I want a Servers view within the IDE so that I can manage and monitor local or remote servers directly from my development environment.
As a developer, I want to add, configure, and remove server instances (e.g., local web servers) so that I can test my applications against different server environments.
As a developer, I want to start, stop, and restart servers from the Servers view so that I can quickly control server state without leaving the IDE.
As a developer, I want the Servers view to display real-time server status (e.g., running, stopped, error) so that I can monitor server health at a glance.
As a developer, I want server console output/logs accessible from the Servers view so that I can debug issues without switching to an external terminal.
